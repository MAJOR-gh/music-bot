# Changelog

## v1.0 — первый релиз 🎵

**Chernarus FM** — музыкальный бот для Discord. Стримит аудио с YouTube и
SoundCloud прямо в голосовой канал, с очередью, кнопками управления и живой
панелью «сейчас играет». Аудио идёт напрямую, без скачивания на диск.

### ▶️ Воспроизведение
- **`/play <ссылка или текст>`** — основная команда.
  - **Ссылка** (YouTube/SoundCloud) — играет сразу.
  - **Текст** — бот ищет **топ-5 вариантов** и показывает выпадающий список с
    названием и длительностью. Удобно, когда у трека есть версии *longmix,
    sped up, slowed, remix* и т.д. — выбираешь нужную сам.
- **Очередь на каждый сервер.** Добавляй треки во время игры — они встанут в
  очередь и заиграют по порядку, ничего не прерывая.

### 🎛️ Живая панель-плеер
- Под играющим треком — панель с кнопками: ⏸️ **Пауза** · ▶️ **Продолжить** ·
  ⏭️ **Пропустить** · ⏹️ **Стоп**.
- Панель **обновляется в реальном времени** при смене трека, паузе и пропуске:
  показывает название, длительность, автора, кто заказал, обложку и блок
  **«Далее в очереди»** (топ-5).
- Кнопки и слэш-команды синхронизированы — что бы ты ни нажал, панель отражает
  актуальное состояние.

### ⌨️ Команды
| Команда | Действие |
|---|---|
| `/join` | Подключить бота к твоему голосовому каналу |
| `/leave` | Отключиться и очистить очередь |
| `/play <запрос>` | Играть по ссылке или искать по тексту (выбор из вариантов) |
| `/skip` | Пропустить текущий трек |
| `/pause` · `/resume` | Пауза / продолжить |
| `/stop` | Остановить и очистить очередь (бот остаётся в канале) |
| `/queue` | Показать очередь |
| `/nowplaying` | Что играет сейчас |

### 🤖 Автоматика
- ⏱️ Авто-отключение через 5 минут бездействия (настраивается `IDLE_TIMEOUT`).
- 👥 Авто-отключение, когда в канале не осталось людей.
- 🔒 Защита от запуска нескольких плееров на один сервер.
- 🔁 Авто-переход к следующему треку и авто-reconnect потока при обрывах.

### 🛠️ Под капотом
- **discord.py 2.7** (поддержка E2EE/DAVE-голоса) + **yt-dlp** + **FFmpeg**.
- Поиск вариантов через быстрый flat-режим yt-dlp: поток скачивается только для
  выбранного трека.
- Конфигурация через `.env` (токен, ID сервера, таймаут, путь к FFmpeg).
- Очередь и состояние — отдельные на каждый сервер.
